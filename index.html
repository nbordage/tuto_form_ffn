Tutoriel – Comment remplir le formulaire FFN (PDF)

Ce tutoriel vous guide étape par étape pour remplir correctement le formulaire PDF fourni par la Fédération Française de Natation.

📥 Étape 1 : Télécharger le PDF

Cliquez ici pour télécharger le PDF FFN

🖥️ Étape 2 : Ouvrir le PDF

Utilisez de préférence Adobe Acrobat Reader (gratuit) pour ouvrir le fichier.

✍️ Étape 3 : Remplir les champs

Cliquez sur les zones de texte pour entrer vos informations : nom, prénom, numéro de licence, etc.

💾 Étape 4 : Enregistrer et envoyer

Une fois le formulaire rempli, cliquez sur Fichier > Enregistrer sous et envoyez-le à l’adresse demandée.

❗ Problèmes fréquents

  • Le PDF ne se remplit pas ? Essayez avec Acrobat Reader (évitez le navigateur).
  • Vous ne pouvez pas sauvegarder ? Faites une impression PDF (« Imprimer » > PDF).
